===============================
A Fun Experiment of Tic Tac Toe
===============================

Install
-------

.. code-block::

   pip install TicTacToe4fun


Execute
-------

>>> from TicTacToe4fun import play
>>> game = play()
>>> game.trials()
>>> game.trials(verbosity=1, n_trials=1)
